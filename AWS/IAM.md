# Permissions

Every IAM User should typically have at least these permissions:
- ```AllowManageOwnSecurityCredentials```
- ```IAMUserChangePassword```
- ```AllowManageOwnMFA``` (if you use MFA...)

Useful but benign additions to ```ReadOnlyAccess```.
