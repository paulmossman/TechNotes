# WSL

## Basic commands

   List running distros
   wsl --list --running

   List available Linux distributions
   wsl --list --online
   
   Install a specific Linux distribution
   wsl --install --distribution <Distribution Name>
   
   Terminae a running distro
   wsl --terminate ____
